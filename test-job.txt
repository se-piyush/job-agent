About the Role

We are looking for a skilled and motivated SDE II - Node.js Developer to join our engineering team. The ideal candidate is a strong hands-on engineer with experience building production-grade applications using Node.js and modern backend technologies. You should be comfortable working in a collaborative environment, participating in technical discussions, and delivering features with a focus on quality and timelines.

Design, develop, test, and maintain backend services and APIs using Node.js.
Deliver features and enhancements with a strong focus on quality, performance, and maintainability.
Collaborate with Product Owners, Frontend Engineers, QA, and DevOps teams throughout the software development lifecycle.
Write clean, reusable, and well-tested code following engineering best practices.
Participate in code reviews and contribute to improving code quality across the team.
Troubleshoot and resolve production issues in a timely manner.
Contribute to application performance optimization and reliability improvements.
Participate in technical discussions and contribute to solution design for new features.
Continuously explore and adopt modern engineering practices, including AI-assisted software development.
